# 类比银行 / Analogy Bank

> 每个技术概念配一个生活类比，用作回答的「开场」，降低面试官理解成本。
> 模拟面试评「类比习惯」、教练生成回答、复习卡片都从这里取。
>
> 一个好类比 = 抓住概念的**本质关系**，而不是表面相似。

## 格式

### <概念名>
- **类比**：<一句话生活类比>
- **映射**：<类比里的 A 对应技术里的 B，关键映射点 1-2 个>
- **适用层级**：L1-L2 开场

---

## 示例（删掉换成你自己的）

### 消息队列 / Message Queue
- **类比**：像餐厅的取餐叫号——下单（生产者）和取餐（消费者）解耦，厨房按自己节奏出餐，客人按号来取，互不阻塞。
- **映射**：叫号机 = 队列；下单顾客 = 生产者；取餐顾客 = 消费者；削峰 = 高峰时单子堆在队列里慢慢消化。

### 缓存 / Cache
- **类比**：像把常用的书放在书桌上，而不是每次都去图书馆借——近、快，但桌面放不下全部，且图书馆更新了你桌上的可能是旧版（缓存一致性）。
- **映射**：书桌 = 缓存层；图书馆 = 数据库；放不下 = 容量与淘汰；旧版 = 缓存失效问题。
