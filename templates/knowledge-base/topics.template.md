## L1-L5 通用层级定义

| 层级 | 含义 | 标准 |
|------|------|------|
| L1 | 知道 | 能说出概念、术语 |
| L2 | 懂原理 | 能讲清机制、画出流程 |
| L3 | 工程实战 + 数据 | 有真实项目落地 + 量化结果（**及格线**） |
| L4 | trade-off + 决策 | 能讲权衡、替代方案、踩坑（**资深水平**） |
| L5 | 系统设计 / 跨场景迁移 | 能从零设计、迁移到新场景（**offer 级**） |

统一回答框架（所有技术问题通用）：**背景 → 原理 → 方案 → 权衡 → 数据结果 → 扩展性**
